## Hurtig brug til eksamen

Når du viser et diagram:

1. Peg på subnets og IP-plan.
2. Peg på routere, switches og klienter.
3. Forklar hvilken protokol eller service opgaven handler om.
4. Vis hvordan forbindelsen testes med fx `ping`, `traceroute`, `show route`, Wireshark eller MQTT/SSH commands.
